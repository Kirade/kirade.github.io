Kotlin

Kotlin

Intellij, PyCharm 등으로 유명한 IDE 를 제작하는 JetBrains 에서 만든 프로그래밍 언어입니다. JVM 을 통해서 Java 와 완전하게 호환이 되고 Android 에서 앱 개발시 추천하는 언어로도 유명합니다.

컴파일 및 런타임 환경의 제공, 개발환경 등을 위해 Kotlin 은 JDK 가 개발에 필수적입니다. 따라서 Kotlin 또한 버전별로 JDK 에 버전에 대한 호환 가능여부를 체크해서 사용해야 합니다.
